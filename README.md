# DEX Trade Issue Tracker

## Welcome to the DEX Trade Issue Tracker

Thank you for choosing to contribute to the improvement of DEX Trade. This repository serves as the official issue tracker for reporting bugs, suggesting enhancements, and discussing various aspects related to the DEX Trade xApp and xApp beta, as well as the websites [https://unhosted.exchange](https://unhosted.exchange) and [https://dev.unhosted.exchange](https://dev.unhosted.exchange).

## Getting Started

### How to Report an Issue or Request a Feature

If you encounter any issues or have a feature request, please follow these steps:

1. **Check Existing Issues:** Before submitting a new issue, please check if a similar one already exists.

2. **Create a New Issue:**
   - Click on the "Issues" tab.
   - Click the "New Issue" button.
   - Choose the appropriate template (bug report or feature request).
   - Fill in the details, providing as much information as possible.

3. **Be Descriptive:** Clearly describe the problem or feature you are addressing. Include relevant details such as operating system, browser, and steps to reproduce the issue.

4. **Labels and Milestones:** Use the provided labels to categorize your issue. If applicable, assign the appropriate milestone.

## Responsible Disclosure
